# Notebooks de quantum-core

Esta carpeta contiene notebooks de Jupyter para explorar y visualizar
los algoritmos y circuitos cuánticos del módulo `quantum-core`.

## Contenido

| Notebook | Descripción |
|---|---|
| `01_primer_circuito.ipynb` | Introducción: construir y visualizar el circuito de semáforo cuántico |
| `02_qaoa_optimizacion_trafico.ipynb` | Optimización de tráfico con QAOA MaxCut |

## Cómo ejecutar

```bash
pip install jupyter numpy
cd quantum-core
jupyter notebook notebooks/
```

> **Nota:** Los notebooks están en estado de marcador de posición (*placeholder*).
> Para usarlos con circuitos reales, instala opcionalmente `qiskit` y `qiskit-aer`.

# Notebooks — quantum-core

Este directorio contiene notebooks de Jupyter para explorar el módulo cuántico
de Ciudad Robot de forma interactiva.

## Notebooks disponibles

| Notebook | Descripción |
|---|---|
| `01_primer_circuito.ipynb` | Introducción a los circuitos cuánticos con el semáforo |
| `02_qaoa_optimizacion_trafico.ipynb` | Optimización de tráfico con QAOA |

## Requisitos

```bash
pip install numpy jupyter
# Opcional para hardware real:
# pip install qiskit qiskit-ibm-runtime
```

## Cómo ejecutar

```bash
cd quantum-core
jupyter notebook notebooks/
```

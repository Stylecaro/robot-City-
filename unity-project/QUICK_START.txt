==============================================
PRIMEROS 3 PASOS (TOMA 15 MINUTOS)
==============================================

PASO 1: Crear escena BattleArena
  - Project → Assets/Scenes → Click derecho
  - Create → Scene → Nombrar "BattleArena"
  
PASO 2: Crear terreno 5km x 5km
  - Hierarchy → Click derecho
  - 3D Object → Terrain
  - Inspector → Settings → Width: 5000, Length: 5000

PASO 3: Crear jugador con movimiento
  - Hierarchy → Create Empty → Nombrar "Player"
  - Add Component → Character Controller
  - Add Component → PlayerMovement
  - Agregar cámara como hijo

LISTO! Presiona PLAY (▶️) para probar
